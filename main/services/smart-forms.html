Functions

  • Functions are small scripts that run at the build and deploy times.

  • When we make a change to your site the site gets rebuilt and redeployed. So for example, if you make a blog post the site gets rebuilt and sent to the sever to be distributed to the internet.

SEO Functions

  • This will automatically build or rebuild a site map of your website. Search Engines use the site map to better crawl your site.

  • This function submits recenlty build site maps to Google, Bing and Yandex and lets them know the site has changed and needs to be reindexed.

  • This function runs Google's Lighthouse test on the website. Lighthouse is a testing system Google created to ensure best standards and practices are being used

Code Functions

  • This funtion is basically like spell check for code. It ensures the code I write is valid and working before deploying the site

  • The code minification functions removes all spaces and paragraphs to the code. Making it more machine readable and, smaller.
